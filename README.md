# Minimal TODO

A minimalist TODO application built with a focus on aesthetics and productivity. Featuring a sleek flat design, smooth micro-animations, and powerful keyboard shortcuts.

[Live Demo](https://todo.imdeepmind.com/)

## Features

- **Premium Aesthetics**: Clean, modern interface with a custom-curated color palette.
- **Dark Mode**: Seamless transition between light and dark themes with a dedicated toggle.
- **Smooth Animations**: Interactive micro-animations for adding, deleting, and completing tasks.
- **Local Persistence**: Your tasks are saved automatically to your browser's local storage.
- **Responsive Design**: Works beautifully across all screen sizes.

## Productivity Shortcuts

Master your task list without touching the mouse:

|   Key    |                              Action                                |
|----------|--------------------------------------------------------------------|
|   `a`    | **Add Task** - Opens the task creation modal with auto-focus.      |
|   `c`    | **Clear All** - Removes all tasks with a smooth fade-out effect.   |
|  `1-9`   | **Toggle Task** - Mark the 1st through 9th task as done/undone.    |
|  `Esc`   | **Close** - Discard the current task creation and close the modal. |
|  `Enter` | **Confirm** - Add the task while inside the creation modal.        |

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Start organizing your day!
